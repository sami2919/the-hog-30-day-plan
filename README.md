# The Hog 30 Day Plan

A one-page operator memo for Hudson Liao at The Hog.

## Run Locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Build

```bash
npm run build
npm run start
```

## Deploy on Vercel

Deploy this repository with the default Next.js settings.

The page is built with Next.js, React, and Tailwind CSS. It has no backend and no environment variables.
